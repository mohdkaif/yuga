== Description ==

Have a blog or website in cricket niche? This is exactly the plugin you need. An extremely simple cricket score plugin. It will create a widget that you can drag and drop at any of the sidebars, you have in your theme. You can customize the color, look and feel of the widget so as to match your theme.  The score is fetched from CricBuzz RSS feed and hence are always the latest. It adds color pickers so you can customize the look and feel of the widget in the way you need. 

It shows score of the current match being held anywhere between any international teams. Even if there are multiple matches, it wil show the score of all of them.  



= Features  =

*  Extremely easy to use

*  Score widget

*  Score updates every 15 seconds

*  Customizable theme, easy color pickers

== Installation ==



0. Upload `live_cricket_score` to the `/wp-content/plugins/` directory

1. Activate the plugin through the 'Plugins' menu in WordPress

2. In wordpress backend go into Appearance-> Widgets

3. Drag 'Live Cricket Score' widget and drop it to any of the sidebars

4. Change colors according to your need.
