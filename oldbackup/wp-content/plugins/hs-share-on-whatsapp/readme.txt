== Installation ==

1. Upload the plugin folder `HS Whatsapp Share` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins -> Installed plugins -> Activate HS Whatsapp Share" menu in WordPress.
3. Find Menu for HS Whatsapp Share Plugin in -> Settings options.
